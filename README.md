# ODO Motors - Premium Car Dealership Website

A modern, responsive website for ODO Motors - Uzbekistan's premier car dealership specializing in imported vehicles and used car marketplace.

## Features

- **Imported Cars Section**: Browse premium vehicles imported from international markets
- **Used Car Marketplace**: Buy and sell quality pre-owned vehicles in Uzbekistan
- **Advanced Filtering**: Search and filter cars by brand, price, year, and more
- **Responsive Design**: Fully responsive layout that works on all devices
- **Contact Form**: Easy-to-use contact form for inquiries
- **Modern UI**: Clean, professional design with smooth animations

## Website Structure

- `index.html` - Main website structure with all sections
- `styles.css` - Complete styling and responsive design
- `script.js` - Interactive features and filtering functionality

## Sections

1. **Home** - Hero section with company branding
2. **Imported Cars** - Showcase of premium imported vehicles
3. **Used Cars Marketplace** - Local used car listings
4. **About Us** - Company information and statistics
5. **Contact** - Contact form and company details

## How to Use

Simply open `index.html` in any modern web browser to view the website. No installation or server setup required.

## Technologies Used

- HTML5
- CSS3 (with CSS Grid and Flexbox)
- Vanilla JavaScript
- Google Fonts (Poppins)

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Customization

To customize the website for your needs:

1. **Update Company Info**: Edit contact details in the footer and contact section
2. **Add Real Car Images**: Replace placeholder images with actual car photos
3. **Modify Car Listings**: Update car cards with your inventory
4. **Adjust Colors**: Modify CSS variables in `styles.css` for brand colors
5. **Connect Backend**: Integrate the contact form with your email service
